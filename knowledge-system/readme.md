# 知识系统 v2.0

一个完整的知识管理和推理系统，整合了UID系统、状态光标系统、寻路搜索和逻辑链解析。

## 特性

### 🎯 四大核心模块
1. **UID系统** - 为每个概念分配唯一标识符，支持一词多义
2. **状态光标系统** - 在知识序列中导航、标记和选择
3. **寻路搜索系统** - 在概念网络中查找路径和关系
4. **逻辑链解析** - 解析符号化逻辑表达式，支持嵌套

### ✨ 主要功能
- ✅ 全局唯一标识符生成和管理
- ✅ 一词多义支持
- ✅ 光标导航和状态标记
- ✅ 图搜索和路径查找
- ✅ 逻辑表达式解析
- ✅ 交互式命令行界面
- ✅ 数据导入导出
- ✅ 完整的类型检查
